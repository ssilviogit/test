207  </tr>
208  </table>
209 </div><div class="memdoc">
210 
211 <p>SPF Installation Environment. </p>
212 <p>By parameter is it possible to include another configuration file named as <code>"spf.settings.{SPF_ENVIRONMENT}.php"</code>. Simply changing the value of <code>SPF_ENVIRONMENT</code>, is it possibile to switch between multiple environment-depended configurations. The file <code>"spf.settings.{SPF_ENVIRONMENT}.php"</code> must be placed into one of the <code>include_path</code> of your PHP, and it's immediately loaded.</p>
