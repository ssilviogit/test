◆ SPF_ENVIRONMENT

const SPF_ENVIRONMENT

SPF Installation Environment.

By parameter is it possible to include another configuration file named as "spf.settings.{SPF_ENVIRONMENT}.php". Simply changing the value of SPF_ENVIRONMENT, is it possibile to switch between multiple environment-depended configurations. The file "spf.settings.{SPF_ENVIRONMENT}.php" must be placed into one of the include_path of your PHP, and it's immediately loaded.

See also
spf.settings.HOME.php
