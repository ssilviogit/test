spf_send_request (   path,
  params,
  method,
  destFrame,
  ajax,
  ajaxParams 
)

Creates a form and sends it to the URL passed by parameter.

Parameters
pathURL destination (the "action" of the form).
params(optional, default null) Object with name/value attributes
method(optional, default "post").
destFrame(optional, default null) Set the destination frame as "target". Indicates a DOM element ID to append the form to. If not specified, the form is appended to the body of the document.
ajax(optional, default false) Bollean value. If true, send an ajax request.
ajaxParams(optional, default null) In case of ajax request, this object contains the ajax parameters:
  • "success": The name of the function to be executed in case of success.
  • "error": The name of the function to be executed in case of error.
